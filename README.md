# Quant Link Portfolio

Work-in-progress rebuild.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Deployment

This project is now deployed via Vercel. Local-only changes can be made without pushing.
